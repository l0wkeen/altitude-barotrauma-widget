# altitude-barotrauma-widget

고도 변화 측정으로 **Ear Barotrauma 예방 알림**을 제공하는 안드로이드 홈 화면 위젯 앱

## 주요 기능

- 기압 센서(TYPE_PRESSURE)로 해발고도 실시간 측정
- **5분 누적 변화량** 기반 위험도 판단 (단순 직전 비교 → 의학적으로 정확)
- 위험 단계별 행동 추천 (발살바 / 하품 / 물 마시기 / 정상)
- WorkManager로 서비스 재시작 보장 (15분 주기)
- 부팅 후 자동 재시작 (BootReceiver)
- 기압 센서 없는 기기 명시적 오류 표시

## 위험 단계 기준 (5분 누적 고도 변화)

| 변화량 | 단계 | 추천 행동 |
|--------|------|-----------|
| ±50m 이상 | 위험 🔴 | 발살바 (코 막고 후풍) |
| ±30m 이상 | 주의 🟠 | 하품 또는 침 삼키기 |
| ±15m 이상 | 경미 🔵 | 물을 조금 마시세요 |
| ±15m 미만 | 정상 🟢 | 이상 없음 |

## 빌드 환경

- minSdk: 26 (Android 8.0)
- targetSdk: 35
- Kotlin 1.9.0
- WorkManager 2.9.1
